## Overview

The OpenL Tablets MCP Server enables Claude AI to interact with OpenL Tablets Business Rules Management System through the Model Context Protocol (MCP). This allows Claude to:

- List and manage repositories and projects
- Create, read, update, and delete rules/tables
- Execute rules with test data
- Run project tests
- Deploy projects to production
- Track project history and versions
